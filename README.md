# Animation-rendering-

Create 3d rendering that can be keyed out for background green screen.

## Hour + rating animation creator

Open `animation_creator.html` in a browser.

- The canvas background is pure green (`#00FF00`) for chroma keying.
- Set start and end hour values to animate hour counter transitions.
- Add ratings using `label,value` format (0-5 range) to animate multiple rating bars.
